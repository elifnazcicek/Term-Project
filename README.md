# CSE101 Term-Project: Restaurant Table & Order Manager

## Installation
- Python 3.10+
- No external dependencies

## How to Run
python main.py

## Roles
- Host: assign and release tables
- Server: take orders and close bills
- Manager: menu editing (future extension)

## Persistence
Data is saved in /data as JSON files.

## Testing
Run:
pytest

## Sample Workflow
1. Assign table
2. Open order
3. Add items
4. Send to kitchen
5. Close bill
